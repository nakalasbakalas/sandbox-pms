## 💡 Key Workflows

### Check-In (< 45 seconds)
1. Select arrival from list or search guest
2. Assign room (auto-suggests available clean rooms)
3. Verify guest details
4. Collect payment (PromptPay QR or other methods)
5. Complete check-in → Room instantly appears occupied

### Check-Out (< 60 seconds)
1. Select departure or click occupied room
2. Review folio charges
3. Collect final payment
4. Print receipt
5. Complete checkout → Room marked dirty for housekeeping

### Create Reservation
1. Click "New Reservation" or press `Cmd+N`
2. Select dates and room type
3. Enter guest information
4. Choose source (Direct, OTA, Walk-in)
5. Set rate and deposit
6. Save → Appears on board timeline

### Housekeeping Workflow
1. Open housekeeping view (mobile/tablet)
2. View priority queue (checkouts first)
3. Select room and mark "Cleaning"
4. Complete cleaning
5. Mark "Clean" → Front desk sees update instantly
