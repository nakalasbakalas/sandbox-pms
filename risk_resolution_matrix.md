# Risk resolution matrix

| Risk area | Confirmed evidence | Resolution | Regression coverage |
| --- | --- | --- | --- |
| Payment webhook idempotency under concurrent delivery | `payment_events` has a unique index on `(provider, provider_event_id)`, but `_apply_provider_event()` previously checked for duplicates before insert without handling concurrent insert races | Added nested transaction + `IntegrityError` handling so the second concurrent insert is treated as a duplicate, not a failing webhook | `sandbox_pms_mvp/tests/test_phase9_hosted_payments.py::test_postgres_webhook_processing_remains_idempotent_under_concurrent_calls` |
| Hardcoded fallback nightly rate | `nightly_room_rate()` previously fell back to literal `750.00` when no rule matched | Added `hotel.base_rate` app setting seed and used it as the fallback base rate | `sandbox_pms_mvp/tests/test_phase10_admin_panel.py::test_rate_rules_use_configured_base_rate_and_apply_long_stay_after_fixed_override` |
| Centralized setting validation gap | `upsert_setting()` validated booleans/integers/decimals, but time and key-specific hotel settings were not normalized centrally | Added time validation plus central validation for hotel currency, contact email, and contact phone; property save now preserves `time` value types | `sandbox_pms_mvp/tests/test_phase10_admin_panel.py::test_upsert_setting_centralizes_validation_for_typed_property_fields` |
| Public payment return labeling clarity | Deposit return view did not surface the request kind label, causing incomplete translated context | Added translated request-kind label rendering on the public payment return surface | `sandbox_pms_mvp/tests/test_phase4_public_booking.py::test_public_payment_return_renders_translated_labels` |
| Seed-sensitive dashboard/public-booking regressions | Several tests assumed zero seeded demo reservations in `seed=True` databases | Updated assertions to verify the audited records are present and counts move correctly relative to the existing seeded baseline | `sandbox_pms_mvp/tests/test_phase19_dashboards.py`, `sandbox_pms_mvp/tests/test_phase4_public_booking.py` |
